Live solver. Edit the spot and hit Solve — the inputs are sent to a local server (solve_server.py) that runs solve_spot.py and returns the exact solution: range-form CFR+ over the razz / Stud 8 betting tree (resolve.py), reporting the equilibrium strategy, hero EV (chips), and best-response exploitability. This tool solves node-locked spots — keep ranges narrow (a handful of explicit holdings) so each solve finishes in <1s.
Start the server: cd solver/neural && python3 solve_server.py  (default http://127.0.0.1:8000).
