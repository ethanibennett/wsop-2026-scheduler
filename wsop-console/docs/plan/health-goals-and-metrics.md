# Health Goals & Metrics
*Baseline → target → retest at each phase boundary · tied to the WSOP 2027 year*

The rule for all five: establish a **baseline** in Phase 1 (weeks 1–2), set a **target**, and **retest at each phase boundary** so progress is a number you can watch, not a feeling.

---

## 1. Weight — 180 → 150

- **Target check:** at 5'7", 150 is a healthy, mid-normal goal (BMI ~23.5, down from ~28). Sensible, not aggressive.
- **Rate:** ~1 lb/week, steady — driven by the nutrition defaults + daily movement floor, not a crash cut.
- **Phasing (the strategic part):** front-load the loss into the calm phases; be at goal *before* the grind. No deficit during the grind or WSOP — running low while playing long solo sessions taxes decisions and recovery.
  - End of Phase 1 (late Sept): ~170
  - Year-end: ~157
  - **150 by the start of the grind season (late Feb)**
  - Phases 4–5: maintain 150
  - Phase 6: hold
- **Track more than the scale:** you're building muscle at the same time, so the scale understates fat loss. Watch waist measurement, how clothes fit, and the strength benchmarks. A flat scale during a strength block is not a stall.

---

## 2. Cardiovascular fitness

- **Baseline:** resting HR (Oura), VO2max estimate (Oura/watch), one repeatable field test (1.5-mile or 5K time).
- **Target shape:** resting HR trending down into the athletic range, VO2max up a category, field test faster each retest. *Candidate milestone:* run a continuous 5K comfortably, then push the time down or extend toward 10K. Lock exact numbers once baselined.
- **Retest:** each phase boundary.

---

## 3. Strength

- **Baseline:** max push-ups, max pull-ups (or dead-hang time), plank hold, bodyweight squat reps. Add key barbell lifts if you train with weights.
- **Target shape:** *candidate milestones* — 40+ push-ups, 10 pull-ups, 2-min plank, plus barbell numbers set off the baseline. Lock targets after week-1 testing.
- **Retest:** each phase boundary.
- **Note:** paused during hernia recovery — see §6. Plan the block around it.

---

## 4. Flexibility

- **Baseline:** sit-and-reach / toe-touch, deep-squat hold (seconds you can sit comfortably), shoulder reach (hands behind back, both sides).
- **Target:** flat-foot toe touch, 60s+ relaxed bottom-squat hold, hands meeting behind the back on both sides.
- **Retest:** each phase boundary.

---

## 5. Sleep — via Oura

- **Levers, in priority order:** consistency (bed-2 / wake-10 held tight — the single biggest mover) → duration (~7–7.5 hrs actual) → efficiency + latency (the wind-down ritual drives these).
- **Year-long trend targets:** HRV trending up, resting HR trending down, a stable Sleep/Readiness range.
- Watch the rolling trend, not any single night.

## How it ties together

- Weight loss is front-loaded; the performance phases (grind, WSOP) run at maintenance.
- The hernia repair is the one medical item that can stall the strength goal — doing it early protects both.
- Everything retests at phase boundaries, and the numbers feed the Sunday review (and, later, the tracker view in the app).
